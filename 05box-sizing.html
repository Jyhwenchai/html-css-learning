父子元素不同 box-sizing 情况下的布局表现

  • 当 box-sizing 为 content-box 时，box 的 width = border + padding + content的宽度
  • 当 box-sizing 为 border-box 时，box 的 width = content的宽度，border 和 padding 的宽度会从 content的宽度中扣除

下面统一称垂直于主轴方向的轴为横轴。主轴可能是水平方向或垂直方向的(默认是水平方向也就是从左往由布局)，那么对应的横轴就是垂直方向或水平方向。
  • align-items: 控制横轴的对齐方式
  • align-self: 控制横轴的 flex-item 的对齐方式
  • align-content: 控制所有 flex-item 在横轴空间上的布局方式(注意不是对齐方式)。
  • justify-content: 控制主轴方向的对齐方式
  • gap,gap-column,gap-row: 用于在 flex-item 之间主轴和横轴上的间隙。

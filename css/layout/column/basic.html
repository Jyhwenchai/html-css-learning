Multiple Column

  • column-count: 定义将页面分割为多少列
  • column-width: 定义列的宽度, 当不指定 column-count 时浏览器会自动根据内容的多少来分割为多列
  • column-gap: 定义列之间的间隙
  • column-rule: 定义 column 的样式, 是column-rule-width、 column-rule-style、column-rule-color 的简写
  • column-span: 在指定的某一列中设置，指定该列可跨多列

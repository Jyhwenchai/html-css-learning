Box Model

1. 如果 box 被定义为一个 block，它将拥有以下行为

  • box 将会换行
  • box 将沿内联方向(也就是水平方向)延伸以填充其容意味着盒子将变得与其容器一样宽，填充 100% 的可用空间。
    • 示例：
    div 的宽度占据容器宽度的 100%
  • width 和 height 属性将被考虑。
  • padding、margin 和 border 会导致其他元素被推离框
Box Width / Height 的计算

box 的 width = width + padding-left + padding-right + border-left + border-right

Normal

2. 如果 box 被定义一个 inline，它将拥有以下行为

  • box 不会换行
  • Width 和 Height 将不会生效
  • 垂直的 padding，marign 和 border 将会被应用，但是不会导致其它的 inline box 远离该 box
  • 水平的 padding，marign 和 border 将会被应用，但是不会导致其它的 inline box 远离该 box
    • 示例：
    LeftRight会在同一行显示
Box Width / Height 的计算

box 的 width = width + padding-left + padding-right + border-left + border-right

2. 如果 box 被定义一个 inline-block，它将拥有以下行为

  • box 不会换行
  • Width 和 Height 将生效
  • 垂直的 padding，marign 和 border 将会被应用，但是不会导致其它的 inline box 远离该 box
  • 水平的 padding，marign 和 border 将会被应用，但是不会导致其它的 inline box 远离该 box
Box Width / Height 的计算

box 的 width = width + padding-left + padding-right + border-left + border-right
